# Yapay Zeka Destekli Siber Güvenlik Tarama Aracı

## Kurulum
pip install -r requirements.txt

## Kullanım
python3 main.py

## Modüller
- M1: Nmap Port Tarama
- M3: Web Header Analizi

## Gereksinimler
- Python 3.x
- Nmap
- Groq API anahtarı
